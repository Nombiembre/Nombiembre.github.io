Glosario Topologias de Red

Monotarea

Todos los dispositivos están conectados a un único cable o canal.

Multitarea

Todos los dispositivos están conectados a un concentrador o switch.

Multiusario

Los dispositivos están conectados en una cadena, en la que cada dispositivo está conectado al siguiente y al anterior.

Monousuario

Todos los dispositivos están conectados entre sí, formando una red completa.

Real time

Los dispositivos están conectados en una estructura jerárquica, con un dispositivo raíz que conecta a todos los demás.

No real time

Es una combinación de dos o más topologías de red.

Sistema operativo de propósito general

Una red que cubre un área pequeña, como un edificio o una oficina.

Sistema operativo de propósito específico

Una red que cubre un área grande, como una ciudad o un país.

Sistema operativo de tiempo compartido

Una red que cubre un área intermedia entre una LAN y una WAN.

Sistema operativo multiprocesador

Una red que cubre un área muy pequeña, como una habitación o un vehículo.

Sistema operativo distribuido

Una red que no utiliza cables para conectar los dispositivos.

Sistema operativo de núcleo monolítico:

Una red que utiliza cables para conectar los dispositivos.

Sistema operativo de núcleo microkernel

Una red que envía los datos en paquetes individuales.

Sistema operativo empotrado

Una red que establece una conexión dedicada entre dos dispositivos antes de enviar los datos.

Sistema operativo de servidor

Una red que utiliza etiquetas para identificar los paquetes de datos.

Sistema operativo de escritorio

Una red que utiliza el ancho de banda disponible para enviar los datos. Sistemas operativos
